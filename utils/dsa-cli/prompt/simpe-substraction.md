This is  a simple substraction example.

```js
import { prompt } from 'dsa-cli';
const a = prompt('Enter a number: ');
const b = prompt('Enter another number: ');
console.log(a - b);
```

```bash
$ dsa run simple-substraction.js
Enter a number: 10
Enter another number: 5
```
