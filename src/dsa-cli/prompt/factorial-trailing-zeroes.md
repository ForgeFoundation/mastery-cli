### Description



### Example 1

```bash
Input: n = 3
Output: 0
Explanation: 3! = 6, no trailing zero.
```

### Example 2

```bash
Input: n = 5
Output: 1
Explanation: 5! = 120, one trailing zero.
```

### Example 3

```bash
Input: n = 0
Output: 0
```


### Constraints:

- 0 <= n <= 104
